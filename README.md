# Sales Performance Analytics Dashboard

### 🎯 Objective
To analyze sales trends, identify top-performing products, and visualize key business metrics for better decision-making.

### 🛠 Tech Stack
- **Data Visualization:** Tableau / Power BI
- **Data Processing:** Python (Pandas) / SQL
- **Documentation:** MS Excel

### 📈 Key Insights
- Identified a 20% growth in sales during the holiday season.
- Pinpointed underperforming regions to optimize marketing spend.
- Visualized Profit vs. Revenue across different product categories.
